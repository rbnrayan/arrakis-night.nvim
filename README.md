# arakis-night.nvim

Port / re-adaptation of [fleet dune theme](https://github.com/Anvell/fleet-dune-theme)
(variant: arakis-night) made by Denis T, in lua for neovim.

## Installation

Install via a package manager:

```lua
-- Packer:
use 'rbnrayan/arakis-night.nvim'

-- Lazy
{
  'rbnrayan/arakis-night.nvim',
  lazy = false,
}
```
